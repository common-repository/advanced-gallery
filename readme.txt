Pixieshot Gallery is a powerful gallery plugin built for creating beautiful and mobile-responsive galleries in minutes.

== Description ==

Pixieshot Gallery plugin is a feature-rich gallery plugin for creating beautiful and mobile-responsive galleries within a couple of minutes. The plugin includes 4 Elementor widgets for creating galleries.

Masonry Gallery
Justified Gallery
Grid Gallery
Filterable Gallery

This plugin is ideal for photographers, artists, studios and others who are looking to add attractive galleries to their website.

== Installation ==

1. Upload `advanced-gallery.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
